Overview

RAKS CSV FileSync is a robust ETL (Extract, Transform, Load) utility designed to map and transform CSV files efficiently. It allows you to process multiple source files, map fields visually, and generate compliant output files for downstream systems.

How to Use

1. Upload & Discovery

  • Go to the Upload tab.
  • Option 1 (ZIP): Upload a ZIP file containing your CSVs. The tool creates a secure workspace.
  • Option 2 (Direct): Select multiple CSV files from your local machine (Desktop/GUI mode only).

2. Creating Mappings

  • Navigate to the Mapping tab.
  • Select a Source File from the dropdown.
  • Enter a Target File Name (e.g., output.csv) or select from history.
  • Click Add Field Mapping. Select the source field and type the target field name.
  • Use Auto-Fill to automatically map fields with matching names!

3. Managing Configurations

  • Save: Click "Save Configuration" (or Download Config) to export your work to a .json file.
  • Import: Drag and drop a previously saved JSON config to restore your session.

4. Execution

  • Switch to the Execute tab.
  • The configuration loads automatically.
  • Click Execute Transformation.
  • Download the results as a ZIP file.
