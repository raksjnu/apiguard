MuleGuard - Help & Documentation

MuleGuard is a versatile and comprehensive static analysis solution designed specifically for MuleSoft applications. It is highly flexible and capable of validating any component of your integration, including flow.xml, global configurations, dataweave.dwl files, and more. MuleGuard can be easily configured to support countless rules, tailored to meet your organization's specific needs for standard or customized code review and validation.

üìã What MuleGuard Validates

MuleGuard provides thorough analysis across all core project components:

  • Maven configurations: pom.xml validation
  • Mule runtime manifests: mule-artifact.json structure
  • Mule flows & sub-flows: XML configuration files
  • DataWeave scripts: .dwl files
  • Environment-specific property files: .properties and .policy files

üìö Supported Validation Checks

MuleGuard supports a wide range of validation types to ensure the integrity and quality of your applications. Possible checks include:

  • Structure & configuration: Validate XML flows, flow.xml, global configurations, and file structures.
  • Dependencies: Verify Maven pom.xml configurations and dependencies.
  • Code Content: Search for specific tokens, patterns, or forbidden content within .dwl DataWeave scripts and other source files.
  • XML Validation: Perform deep inspection using XPath, check for specific attributes, and validate element content.
  • Property Management: Validate environment-specific property files, check for mandatory properties, and ensure secure property handling.
  • Custom Logic: Define custom checks for JSON validation, generic token searches, and more to cover any specific organizational requirement.

‚öôÔ∏è Configuration

All rules are configured in src/main/resources/rules/rules.yaml

  • Enable/Disable rules: Set enabled: true/false
  • Severity levels: CRITICAL, HIGH, MEDIUM, LOW
  • Custom parameters: Each rule type supports specific parameters
  • Environment filtering: Target specific environments (DEV, QA, PROD)
