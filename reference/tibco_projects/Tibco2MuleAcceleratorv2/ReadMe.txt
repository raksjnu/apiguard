Accelerator to convert Tibco projects to Mule projects.
Features:
1. Tibco Project to Mule project migration.
2. More than one Tibco project can be migrated in one run.
3. All the As-IS artifacts can be placed from TIbco project folders to Mule project folders.
4. Developer level various checklist included where any deviation is flagged.
