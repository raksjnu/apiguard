API Forge Documentation

Master the art of API Comparison, Baseline Management, and Rapid Prototyping.

API Forge is a flexible, Java-based static and live analysis engine designed for Universal API Compliance. It allows you to validate migrations, compare REST/SOAP responses, and perform baseline testing with deep structural inspection.

üíª Tech Stack

Built with enterprise-grade stability and modern web standards:

Java 17 (LTS) Maven 3.8+ Spark Java Apache HttpClient Jackson (JSON) XMLUnit (SOAP) Vanilla JS/CSS

üöÄ How to Use

  1. Global Setup: Set your Working Directory in Utilities for baseline storage.
  2. Configure: Enter your Endpoint URLs and Payload in the Main panel.
  3. Tokens: Add iteration tokens (e.g., id=100;101) to test multiple scenarios.
  4. Execute: Click Run Comparison and inspect the highlighted structural diffs.
