About GitAnalyzer

Enterprise-Grade Semantic Code Analysis

GitAnalyzer is an advanced comparison engine designed to understand the intent of code changes, not just textual differences. Unlike traditional diff tools that flag extensive false positives due to formatting, attribute reordering, or noise, GitAnalyzer focuses on what matters: specific logic, configuration, and structural integrity.

Key Capabilities
Semantic XML/JSON Compare

Intelligently compares structured data.
<tag a="1" b="2"/> is treated as identical to <tag b="2" a="1"/>. This eliminates noise in migration audits where tools may reorder attributes.

Noise Reduction

Define powerful Regex patterns to ignore irrelevant changes like timestamps, generated IDs, or comments, allowing you to focus on actual regressions.

Multi-Repo Awareness

Simultaneously analyzes paired repositories (e.g. Code vs Configuration) to ensure total environment consistency across releases.

Audit-Ready Reporting

Generates comprehensive, immutable reports with categorized severity (Critical, Modified, Config), suitable for compliance and release sign-offs.

Status Legend & Colors

Badge / Indicator Description
CODE File belongs to the main source code (Logic/Flows). Color: Dark Cyan
CONFIG File belongs to configuration (properties, yaml, pom). Color: Light Blue
MODIFIED File exists in both but content changed. Color: Blue
NEW File exists only in the Featured/Target branch.
DELETED File removed (exists only in Base). Color: Red
CRITICAL High severity change requiring review. Color: Orange-Red

Comparison Analysis Tool

Compare two branches (Base vs Head) to see what changed.

  • Repositories: Auto-fetches available repos from the configured directory.
  • Branches: Select Base (Source) and Head (Target) branches.
  • Smart Compare: Ignores whitespace and designated patterns (e.g., `app.home`).

Git Search

Search for code patterns using Regex across repositories.

  • Local Mode: Search within downloaded repositories.
  • Remote Mode: Search directly on GitLab using API (Requires Token).

Bulk Download

Download multiple repositories simultaneously based on Group or Regex Filter.

Useful for setting up a workspace for the first time.
